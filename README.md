# Игра "Быки и Коровы"

# Описание проекта

Компьютер загадывает 4-значное число а пользователь должен угадать его, получая подсказки в виде "быков" и "коров" и проверки числа с загаданным

# Правила игры
- Бык - цифра правильная и находится на правильной позиции
- Корова - цифра правильная, но находится на неправильной позиции


# Запуск игры
chmod +x game.sh
./game.sh

#Как играть
1. Программа загадывает 4-значное число
2. Введите вашу 4-значную цифру
3. Получите результат:
   Количество быков (правильные цифры на своих местах)
   Количество коров (правильные цифры не на своих местах)
   Проверка числа для упрощения
5. Продолжайте угадывать, пока не получите 4 быка
